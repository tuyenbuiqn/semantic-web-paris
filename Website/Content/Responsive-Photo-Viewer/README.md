# PhotoViewer

PhotoViewer is a JS plugin to view images just like in windows. 

## Main Features

- Modal draggable.
- Modal resizable.
- Modal maximizable.
- Image movable.
- Image zoomable.
- Image rotatable.
- Keyboard control.
- Fullscreen showing.
- Multiple instances.
- Browser support.
